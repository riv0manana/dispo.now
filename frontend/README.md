# dispo.now Frontend

This is the frontend for dispo.now, built with:
- Vite
- React
- TypeScript
- Tailwind CSS (v4)
- TanStack Router
- TanStack Query

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Structure

- `src/routes`: Contains the route definitions and components.
  - `index.tsx`: Landing page.
  - `dashboard/`: Dashboard routes (Layout, Project List).
- `src/components`: Reusable components.
- `src/lib`: Utilities and API client.
